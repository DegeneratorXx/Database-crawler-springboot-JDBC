Available API Endpoints

  • GET /api/metadata - Get complete database metadata
  • GET /api/metadata/tables - Get all tables metadata
  • GET /api/metadata/tables/{tableName} - Get specific table metadata
  • GET /api/metadata/models - Generate model classes for all tables
  • GET /api/metadata/models/{tableName} - Generate model class for a specific table
  • GET /api/config/current - Get current database configuration
  • POST /api/config/upload - Upload custom database configuration
  • POST /api/config/update - Update configuration using direct JSON
  • GET /api/config/test-connection - Test the database connection

JSON Configuration Example

To update configuration directly with JSON, send a POST to /api/config/update with the following structure:

{
  "url": "jdbc:mysql://localhost:3306/your_database",
  "username": "your_username",
  "password": "your_password",
  "modelPackage": "com.dbmetagen.app.model",
  "outputDirectory": "generated"
}
